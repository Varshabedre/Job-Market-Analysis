# 📊 Job Market Analysis Project

## 🔍 Overview
This project analyzes job market trends using Python (Pandas, Matplotlib, Seaborn).

## 📌 Features
- Data Cleaning
- Skill Extraction (Python, SQL, Excel, Power BI, Tableau)
- Visualization of in-demand skills
- Location-based job analysis

## 📊 Output
- Bar chart showing most demanded skills
- Cleaned dataset for further analysis

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn

## 🚀 Conclusion
SQL and Excel are the most in-demand skills, followed by Python and Tableau.
